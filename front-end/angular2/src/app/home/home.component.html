PHP REST API !

The application back-end is built in PHP using Laravel 5 framework. The front-end is built in Angular 2 using TypeScript.!

  • To run the php server goto the root of the folder and in terminal type php artisan serve.
  • Mysql configuration is required to access the database.
  • The .env file has DB_CONNECTION=mysql , DB_NAME=mydb,DB_USERNAME = root, DB_PASSWORD=
  • After configuring the mysql, run php artisan migrate followed by php artisan serve.
    • To run the Angular 2 front-end , dependencies are needed to be installed which can be done by running tthe command npm install.
    ng serve spins up the Angular 2 development server.
    The CRUD REST API uses a Customer model having fields such as id, email and title. The Customer Controller handles the various GET, POST, UPDATE AND DELETE requests.
